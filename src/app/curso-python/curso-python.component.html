Cursos de Python

27/03/2020

Este público es cada vez más grande pero típicamente no tienen tiempo disponible. Con esto en mente, el curso fue diseñado con un formato amigable de cuatro clases de tres horas de forma tal que pueda realizarse como actividad complementaria al estudio curricular o al trabajo, con un fuerte énfasis en ejemplos prácticos que se relacionen fácilmente con casos de aplicación realistas para la vida profesional. A esto se suma la asistencia personalizada para dar respuesta a la gran heterogeneidad de los participantes, considerando las diferentes carreras, profesiones y edades presentes.

Clase 1

 Introducción a la programación en Python, Variables, Operadores, Control de flujo, Funciones

Clase 2

 Tipos de datos (int, float, string, boolean), Estructuras de datos (Lista, Dict, Set, Tuplas), Operaciones con tipos de datos, Operaciones con estructuras de datos.

Clase 3

 Repaso de flujo, Repaso de estructuras de datos, Lectura y escritura de archivos, Operaciones sobre datos de archivos, Introducción a librerías

Clase 4

 Introducción a matplotlib, Visualización de datos, Personalización de los gráficos, Ventajas y desventajas uso de Python y Excel
